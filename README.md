# Projet SQL - Location de Véhicules

Système de gestion de location de véhicules avec requêtes SQL.

## Tables

- **Vehicule** : marque, modèle, autonomie, état, ville
- **Client** : nom, prénom, email
- **Location** : dates, client, véhicule, stations
- **Station** : nom, localisation
